# Nave de papel

Una **comunidad** con la afición a las **naves de papel**, para compartir consejos y modelos.

## Comenzando 🚀

Aquí podrás encontrar modelos de naves para **descargar**, ya sea sin color o con color.

Esperamos que nos ayudes a construir esta comunidad, **contribuyendo y colaborando** con otros usuarios.

## Construido con 🛠️

* HTML, CSS - Stylus y Js
* Programas: [Visual Estudio Code](https://code.visualstudio.com/) - [Prepos](https://prepros.io/)
